# Baseline Notes

- This branch is intentionally orphaned from main.
- Files are placed under initial-source/ to avoid replacing current FUNKY application paths.
- The baseline excludes local repository metadata, generated artifacts, uploads, environment files, secret-key file types, and unsupported absolute symlinks.
- The baseline should be used only to compare initial human-written behavior against current FUNKY main.
- Do not open implementation PRs from this branch.
- If a future audit report is needed, create a separate docs-only PR after human review.
